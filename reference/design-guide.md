# 销售讲义设计规范

参考 [Anthropic pptx skill](https://github.com/anthropics/skills/tree/main/skills/pptx) 的设计原则，确保生成**带颜色、排版、视觉元素**的讲义。

## 配色

**不要用纯白底黑字。** 选用与主题匹配的配色。

| 主题 | 主色 | 辅色 | 强调色 |
|------|------|------|--------|
| **Teal Trust**（金融/支付，默认） | `#028090` | `#00A896` | `#02C39A` |
| Ocean Gradient | `#065A82` | `#1C7293` | `#21295C` |
| Charcoal Minimal | `#36454F` | `#F2F2F2` | `#212121` |

- 主色占 60–70% 视觉权重
- 封面用深色背景，内容页用浅色背景（「三明治」结构）
- 或全深色营造高端感

## 每页必有视觉元素

**纯文字幻灯片容易被忽略。** 每页至少包含：

- 图标（emoji、Font Awesome、SVG）
- 彩色标题栏/色块
- 圆角矩形、分隔线
- 标签高亮（如「极稳通道：xxx」中标签用主色）

## 版式变化

避免每页同一布局。可交替使用：

- 封面：居中大标题 + 副标题 + slogan 标签
- 内容页：左对齐标题栏 + 要点列表
- 对开页：左问题、右方案（若 Markdown 支持）
- 产品页：标题 + 定义 + 核心价值（标签：说明）

## 字体与间距

- 标题：32–44pt 粗体
- 正文：14–18pt
- 页边距：≥ 0.5"
- 内容块间距：0.3–0.5"

## 避免

- 标题下加装饰线（AI 感强）
- 低对比度（浅灰字配浅色底）
- 每页版式完全一致
- 纯白底 + 黑字无任何装饰
